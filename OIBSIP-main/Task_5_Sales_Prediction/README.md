# Sales Prediction using Python

## Project Overview
This project predicts product sales based on advertising budgets spent on TV, Radio, and Newspaper platforms using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- VS Code

## Machine Learning Algorithm
- Linear Regression

## Features Used
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

## Project Workflow
1. Load dataset
2. Data preprocessing
3. Data visualization
4. Train-test split
5. Train Linear Regression model
6. Evaluate model performance
7. Predict sales using user input
