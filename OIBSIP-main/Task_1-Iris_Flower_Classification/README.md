# Iris Flower Classification

## Project Overview
This project is based on Machine Learning classification using the Iris flower dataset. The model predicts the species of iris flower based on different flower measurements.

## Dataset
The Iris dataset contains the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- VS Code

## Machine Learning Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Random Forest Classifier

## Project Workflow
1. Importing libraries
2. Loading dataset
3. Data preprocessing
4. Data visualization
5. Splitting dataset into training and testing sets
6. Model training
7. Model evaluation
8. Prediction of flower species

## Output
The model predicts the species of iris flower with high accuracy.
