# Unemployment Analysis with Python

## Project Overview
This project analyzes unemployment trends in India using Python. The analysis focuses on unemployment rates across different regions and time periods, especially during the Covid-19 period.

## Dataset
The dataset contains unemployment statistics such as:
- Region
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Labour Participation Rate (%)

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Project Workflow
1. Load dataset
2. Data cleaning
3. Check missing values
4. Data visualization
5. Unemployment trend analysis
6. Regional unemployment comparison
7. Heatmap analysis
8. Conclusion and insights

## Visualizations Used
- Bar Chart
- Line Plot
- Heatmap

## Project Insights
- Unemployment rates vary across regions.
- Covid-19 caused a significant increase in unemployment.
- Some regions were more affected than others.
