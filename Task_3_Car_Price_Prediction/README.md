# Car Price Prediction using Machine Learning

## Project Overview
This project predicts the selling price of a car using Machine Learning. The prediction is based on features such as present price, kilometers driven, fuel type, transmission type, ownership, and car age.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- VS Code

## Machine Learning Algorithm
- Linear Regression

## Features Used
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner
- Car Age

## Project Workflow
1. Load dataset
2. Data preprocessing
3. Convert categorical values into numerical values
4. Train-test split
5. Train Linear Regression model
6. Evaluate model performance
7. Predict car selling price
