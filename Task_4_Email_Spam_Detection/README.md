# Email Spam Detection using Machine Learning

## Project Overview
This project detects whether an email or message is spam or not spam using Machine Learning and Natural Language Processing techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- VS Code

## Machine Learning Algorithm
- Multinomial Naive Bayes

## Project Workflow
1. Load dataset
2. Clean data
3. Convert text into numerical vectors
4. Train-test split
5. Train Naive Bayes model
6. Predict spam or non-spam messages

## Features Used
- Email or message text content
